    1. QGC環境のインストール¶

    QGC(QGroundControl)の公式ページにアクセスして、QGC環境を入手します。

    QGROUNDCONTOL公式ページ

    公式ページに行くと、DOWNLOADというボタンがあるのでクリックします。

    DOWNLOADボタンをクリックすると、各OSごとのインストーラが配布されています。Windows用を入手します。

    インストーラをダウンロードできたら、インストーラを起動して、画面に従ってインストールします。
