# 本ドキュメントについて

本ドキュメントは、箱庭ドローンシミュレータとOSSのフライトコントローラ PX4を連動させて、シミュレーションの操作をする方法を解説しています。

対象のPlatformは、**Ubuntu 24.04 LTS**を想定しています。他のPlatformは対象外となります。
