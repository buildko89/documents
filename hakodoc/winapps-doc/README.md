# 箱庭ドローンシミュレータ Windows環境へのインストール手順

箱庭ドローンシミュレータを利用するためにWindows環境で箱庭ドローンシミュレータを操作するアプリケーションソフトウェアのドキュメントとなります。

箱庭ドローンシミュレータ用のアプリケーションソフトウェアは、以下の機能を持ったアプリケーションがあります。各アプリケーションは、箱庭ドローンシミュレータアプリケーション用のインストーラにてWindows環境にインストールされ、利用することができます。

|ドキュメント名|内容|
|:---|:---|
|[箱庭ドローンシミュレータPython APIアプリケーション]()|箱庭ドローンシミュレータが用意しているドローン飛行用のPython APIを利用したドローンの自律航行が体験できるアプリケーションです。|
|[箱庭ドローンシミュレータコントローラ操作アプリケーション]()|箱庭ドローンシミュレータのドローンをゲームコントローラなどで、ドローン操縦を体験できるアプリケーションです。|
