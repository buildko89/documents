# 箱庭でのUnreal Engine利用について

このドキュメントは、箱庭環境でビジュライズ部分で`Unreal Engine`を利用するためのドキュメントになります。

Unreal Engineのバージョンは、5.6.1を利用した場合になります。

# 箱庭 Unreal Engineの入手

箱庭ラボのgithubからUnreal Engine用のアセットを入手します。

```bash
$ mkdir work
$ cd work
```
```bash
git clone --recursive https://github.com/hakoniwalab/hakoniwa-unreal-drone.git
```
