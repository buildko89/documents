# このドキュメントついて

developer-docは、箱庭ドローンシミュレータ関連でビルドなど開発に必要となるドキュメントとなります。

|ドキュメント名|説明|
|:---|:---|
|mujoco-doc|MuJoCoを利用する場合の設定や利用方法について説明|
|px4-doc|OSSのフライトコントローラ PX4の設定や利用方法について説明|
|ubuntu-doc|Ubuntu 24.04上で箱庭ドローンシミュレータを利用する方法について説明|
|unity-doc|箱庭ドローンシミュレータでUnityアプリケーションの設定や利用方法について説明|
|unreal_engine-doc|箱庭ドローンシミュレータでUnreal Engineアプリケーションの設定や利用方法について説明|


## PCスペック(想定)

PCのスペックは、以下のものが最低限必要になります。

- PCのスペック(最低限)

|PCスペック|説明|
|:---|:---|
|CPU|corei7 or Ryzen 7 以上|
|Memory|16Gbyte|
|Storage|500GByte|
|GPU|NVIDIA RTX系 or AMD REDEON系のもの|
